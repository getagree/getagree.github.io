Data analysis pipelines

Reusable pipelines for analyzing real-world datasets in specific domains.

  • bcpm-msaenet - Pipeline for the precisionFDA challenge on brain cancer predictive modeling and biomarker discovery. Features multi-step adaptive SCAD-net and relaxed stability selection for feature selection, and a two-layer stacking tree model with xgboost, lightgbm, and catboost.
  • rankv - Pipeline for the precisionFDA challenge on detecting adverse event anomalies using FDA open data. Features a rank aggregated signal detection method for Vaccine Adverse Event Reporting System (VAERS) data, with base signal detection methods including Gamma Poisson Shrinker, PRR, ROR, and BCPNN. [Website]
